Checked Polyfill

The :checked pseudo-class allows developers to create custom checkboxes and checkbox buttons using pure CSS. This CSS3 feature doesn't work on IE8 or earlier, so on those browsers, Checked Polyfill provides a .checked class that works just like :checked.
